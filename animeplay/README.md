# animeplay app 🤖

AnimePlay is a simple application that features a surface level application of 3B animation in flutter using the package O3D, .glb animated characters, mainly from Ready Player Me, and other sources.


## Project setup 👨🏿‍💻

On launch, the app display a page containing similar animated characters facing each other in a position that looks like they are conversing, we have a control panel section below the characters, one pane of animation for each character,

- Animations include the following:
  - Idle, Walk, Run, Attack, and Fall.
  - Kindly watch this walkthrough video for futher clarity

## Section Two

On the home screen, you'll notice a button at the right end of the appBar, with title 'View Characters'. This brings you to a page that has a few other animated motions, and allows for you to move characters using the arrow keys. See arrow key actions below:

- Arrow keys:
  - Left arrow key: Turns character to the left.
  - Right arrow key: Turns character to the right.
  - Up arrow key: Moves character to center.

- On this next section, we have three animated characters with fictional names, Diloo, Monarchh, and Nanna. All having same set of animations
